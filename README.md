## UX Research / Designer Test

El siguiente desafío, busca conocer más respecto a tus habilidades para mejorar la experiencia de usuario en una funcionalidad básica de la plataforma Envíame.

El desafío consiste en re-diseñar la experiencia del flujo de creación de envío envíos en nuestra web. 

### Entregables:
- Evaluación de usabilidad.
- Blueprint / Wireframe de recomendaciones.

***Siéntente libre de agregar cualquier otro entregable que consideres pertinente.***

* Asegurate de dejar tu trabajo en un repositorio de github público y de enviar el link al mismo a *tech-test@enviame.io*
